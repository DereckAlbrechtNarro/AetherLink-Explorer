# AetherLink Explorer

**Stay connected anywhere on Earth (or Mars)** — Portable satellite internet kits, rugged power stations, off-grid adventure gear & exclusive FSD collectibles.

## Tech Stack
- **Backend**: Node.js + Express + Sequelize + PostgreSQL + Socket.io + Stripe + JWT
- **Frontend**: Next.js 15 (App Router) + Tailwind CSS + React Leaflet + Socket.io-client
- **DevOps**: Docker + Render (backend) + Vercel (frontend)

## Features
- Real-time interactive satellite coverage map with Socket.io live updates
- Full e-commerce flow with JWT authentication and Stripe checkout
- AI-powered mock off-grid route planner
- Responsive, modern UI with Tailwind
- Fully Dockerized backend

## Live Links
- Frontend: https://aether-link-explorer.vercel.app/
- Backend: https://aetherlink-explorer-backend.onrender.com 

## Setup
1. Clone backend & frontend repos
2. Backend: `npm install && npm run dev`
3. Frontend: `npm install && npm run dev`
4. Add products via Postman (see screenshots or Postman collection)
